# Inner Officials — Game Design

A resource and agent management roguelite with meta-progression, set in the Tang Dynasty imperial harem. You play as a concubine of the Wu family entering the Forbidden City. Each run, your achievements increase Wu family standing — when you fall, the next Wu daughter enters with those accumulated benefits.

Heavily inspired by **The Sultan's Game** by Double Cross. The Sultan's Game reference docs serve as a mechanical starting point; the Inner Officials docs are the living design that will diverge and grow independently.
